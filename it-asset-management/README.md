# AI-Powered IT Asset Management System

## 🚀 Overview

This project is a **centralized IT asset management system** enhanced with an **AI-powered chatbot**.

---

## 🔥 Features

* Role-Based Access Control (RBAC)
* Asset allocation & tracking
* Subscription & cost management
* AI chatbot with **role-based response control**
* Secure query validation

---

## 🤖 AI Chatbot Logic

* Admin: Full database access
* User: Limited access based on role
* Unauthorized queries: Automatically rejected

---

## 🧠 Prompt Engineering Used

* Structured prompts for role-based responses
* Context-aware query filtering
* Secure response generation

---

## 🛠 Tech Stack

* Backend: Node.js
* Frontend: Angular
* Deployment: Docker, Nginx
* AI: Prompt Engineering (LLM-based)

---

## 📊 Use Case

Enterprise IT teams for asset tracking and intelligent data access.
